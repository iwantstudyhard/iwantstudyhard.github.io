# Linux 常用命令

> **重要提示**：
>
> 1. 可以先了解shell脚本编辑，之后深入学习
> 2. Linux命令严格区分大小写！
> 3. 命令太长时，可以在行尾输入 `\` 然后按回车，在下一行的 `>` 后继续输入

## 简单命令

| 命令 | 描述 |
|------|------|
| `echo` | 输出字符到终端 |
| `date` | 显示当前日期和时间 |
| `cal` | 显示日历 |
| `clear` | 清空终端信息 |
| `passwd` | 修改密码（输入时不显示，保证安全） |

**`cal` 命令参数说明：**

- 不添加任何参数：显示当前月份的日历
- 只指定年份：显示该年所有月份的日历
- 同时指定月份和年份：显示该月日历

## 文件操作命令

### cat（concatenate）

连接文件并打印到标准输出设备上，主要用于查看和连接文件。

**语法格式：**

```bash
cat [选项] [文件]
```

**常用选项：**

- `-n`：显示行号，在每一行前加上行号
- `-b`：显示行号，但只对非空行进行编号
- `-s`：压缩连续的空行，只显示一个空行
- `-v`：显示一些非打印字符

**使用示例：**

- 显示文件内容：`cat filename`
- 连接文件：`cat file1 file2 > combined_file`
- 创建文件：`cat > filename`（输入内容后按 `Ctrl+D` 保存）
- 显示命令输出：`ls -l | cat`

---

### less 和 more

`less` 与 `more` 类似，可以随意浏览文件，支持翻页和搜索，支持向上翻页和向下翻页。

**语法：**

```bash
less [参数] 文件
```

**常用参数：**

- `-b <缓冲区大小>`：设置缓冲区的大小
- `-f`：强迫打开特殊文件

**less 内部命令：**

- `空格键`：向下滚动一页
- `b`：向上滚动一页
- `d`：向下滚动半页
- `u`：向上滚动半页
- `y`：向上滚动一行
- `回车键`：向下滚动一行
- `[pagedown]`：向下翻动一页
- `[pageup]`：向上翻动一页
- `h`：显示帮助界面
- `Q`：退出 less 命令

## 链接文件命令

### 硬链接

- 没有真正创建文件，只是创建了一个指向原文件 inode 的目录项
- inode 节点的链接数会增加
- **不能对目录创建硬链接**
- **不能跨文件系统创建硬链接**

**相关命令：**

- `ln`：创建硬链接
- `rm`：删除硬链接

### 符号链接（软链接）

- 创建了一个新文件，文件中包含目标文件的路径
- 相当于 Windows 的快捷方式
- 没有硬链接的限制

**相关命令：**

- `ln -s`：创建软链接

## 文件权限管理命令

### 权限类别

- **文件主权限**：文件所有者的权限
- **组用户权限**：文件所属用户组的权限
- **其他用户权限**：其他用户的权限

### 权限类型

- `r`：读取权限
- `w`：写入权限
- `x`：执行权限
- root 用户拥有对所有文件的完全权限

### 相关命令

- `chmod`：更改文件的权限
- `umask`：设置新建文件的默认权限掩码

## 端口占用检查

在 Linux 中可以使用多种方法查看端口是否被占用：

| 方法 | 命令 | 描述 |
|------|------|------|
| **netstat** | `netstat -tuln \| grep :<端口号>` | 使用 netstat 查看 |
| **lsof** | `lsof -i :<端口号>` | 查看端口占用情况 |
| **ss** | `ss -tuln \| grep :<端口号>` | 使用 ss 查看 |
| **netcat** | `nc -zv 127.0.0.1 <端口号>` | 使用 netcat 测试 |
| **fuser** | `fuser <端口号>/tcp` | 查看使用端口的进程 |
