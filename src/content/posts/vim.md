# vi 编辑格式

## vi 编辑的基本命令：（大小写是很区分的！）

 进入文件：
+ Vi 文件名
+ Vi +行号 文件名
+ Vi +/词 文件名

`Vi 文件 1 文件 2  打开多个文件（完成对第一个编辑，用：w 保存，输入：n 进入第二个文件，如果要跳过文件，输入：e，可以跳过）`
## 光标移动：
方向键或者HJKL 分别是⬅⬇⬆➡  
^直接到行首，但是不 insert  
$直接到行尾，但是不 insert  
O 在上一行输入，创建一个新的空白行输入，进入 insert
o 在下一行输入，创建一个新的空白行输入，进入 insert
a     当前光标的后面输入信息
A   当前行的行首输入
i     当前光标的前面输入信息
I     当前行的行尾输入信息

## 退出

：进入行尾模式，可以输入命令
：wq 保存并退出文件
：w 保存修改
：q 退出编辑器
：q！强行退出不保存

## 修改
dd -- 剪切当前行。
yy -- 复制当前行。
p（小写） -- 粘贴剪贴板内容到光标下方。
P（大写）-- 粘贴剪贴板内容到光标上方。
u -- 撤销上一次操作。
Ctrl + r -- 重做上一次撤销的操作。（撤销的撤销）
:set nu 显示行号，设定之后，会在每一行的前缀显示该行的行号
:set nonu 与 set nu 相反，为取消行号！
